# Code of Conduct

## Our Commitment

We are committed to providing a welcoming, inclusive, and professional environment for all contributors to the Actionbook project.

## Our Standards

We encourage behaviors that contribute to a positive environment:

- **Be respectful** - Treat everyone with respect and kindness
- **Be collaborative** - Work together constructively and welcome diverse perspectives
- **Be professional** - Focus on technical discussions and constructive feedback
- **Be inclusive** - Welcome contributors of all backgrounds and experience levels
- **Be helpful** - Support others in learning and growing

We expect all participants to communicate professionally and focus on what is best for the community.

## Scope

This Code of Conduct applies to all project spaces including:

- GitHub repository (issues, pull requests, discussions)
- Project documentation
- Social media channels (X/Twitter, Discord)
- Any other official Actionbook community spaces

## Reporting

If you experience or witness unacceptable behavior, please report it by:

- Opening a private issue on GitHub
- Contacting the project maintainers directly
- Emailing [contact information to be added]

All reports will be reviewed and addressed appropriately.

## Enforcement

Project maintainers are responsible for clarifying and enforcing this Code of Conduct. They may take appropriate action in response to any behavior they deem inappropriate or in violation of community standards.
